## _5.2 산술 연산_

### _5.2.1 표현식과 연산자_

> **1) 산술 이항 연산자**

| 연산자 |     뜻     |    예제    | 예제의 뜻              |
| :----: | :--------: | :--------: | ---------------------- |
| **+**  | **더하기** | **a + b**  | a와 b를 더한 값        |
| **-**  |  **빼기**  | **a - b**  | a와 b를 뺀 값          |
| *****  | **곱하기** | **a * b**  | a와 b를 곱한 값        |
| **/**  | **나누기** | **a  / b** | a를 b에 나눈 값        |
| **%**  | **나머지** | **a % b**  | a를 b로 나눈 나머지 값 |

1. **정수끼리 나누어도 결과가 부동소수점이 된다.**

2. **나머지 연산자 %의 피연산자는 부동소수점이다.**

   - `15 % 4` // → 3
   - `5 % 1.5` // → 0.5

3. **연산자는 피연산자 중 하나가 문자열이면 나머지 피연산자를 문자열로 만든다.**

   - `1 + "2month"` // → "12month" : 숫자 1을 문자열 "1"로 타입을 바꾸고 "2month"와 연결

4. **기타**

   - `0/0` // → NaN : 계산할 수 없음
   - `"one" * 1` // → NaN : 계산할 수 없음
   - `true + true` // → 2 : 논리값의 타입을 숫자로 바꾸고 더함
   - `1 + null` // → 1 : null을 0으로 바꾸어 더함
   - `1 + underfined` // → NaN : underfined를 NaN으로 바꾸어 더함

   

> **2. 산술 단항 연산자**

| 연산자 |             뜻             | 예제                 | 예제의 뜻                                                    |
| :----: | :------------------------: | :------------------- | ------------------------------------------------------------ |
| **++** |      **증가 연산자**       | **++a**<br />**a++** | a에 1을 더한 다음에 a값을 평가한다.<br />a를 평가한 다음에 a에 1을 더한다. |
| **--** |      **감소 연산자**       | **--a**<br />**a--** | a에서 1을 뺀 다음에 a값을 평가한다.<br />a를 평가한 다음에 a에서 1을 뺀다. |
| **+**  | **아무것도 처리하지 않음** | **+a**               | a와 같은 값으로 평가한다.                                    |
| **-**  |       **부호 반전**        | **-a**               | a의 부호를 반전한 값으로 평가한다.                           |

```javascript
a = 1;
b = ++a; // b는 2가 되고 a는 2가 된다.(a에 1을 더하고 a를 b에 대입한다)
c = a++ + 2 // c는 4가 되고 a는 3이 된다.(a + 2를 c에 대입하고 a에 1을 도한다)
```



> **3. 산술 대입 연산자**

| 연산자 |    예제    |    뜻     |
| :----: | :--------: | :-------: |
| **+=** | **a += b** | a = a + b |
| **-=** | **a -= b** | a = a - b |
| ***=** | **a *= b** | a = a * b |
| **/=** | **a /= b** | a = a / b |
| **%=** | **a %= b** | a = a % b |

***EX) 자바스크립트에서 사용하는 64비트 부동소수점 규격(IEEE764)***

![](https://user-images.githubusercontent.com/75871005/124109878-92f4b580-daa2-11eb-9b60-e83449eaf7c7.png)

- 즉, 가수 부분의 유효한 자릿수가 2진수 53자리 이므로 10진수 자릿수는 2^53 = 10^15.95로 약 16자리

## _5.4 논리 연산자와 관계 연산자_

### _5.4.1 관계 연산자_

- 관계연산자는 두 개의 **피연산자**를 비교한 결과를 **논리값(true/false)**로 반환한다.
  - 주로 `제어 구조(if/else 문, while 문, do/while 문, for 문)`에서 조건식을 만들 때 사용

| 연산자  |           뜻           |   예제    | 예제의 뜻                                                 |
| :-----: | :--------------------: | :-------: | --------------------------------------------------------- |
| **==**  |    ***값이 같음***     | `a == b`  | ***a 값과 b 값이 같으면 true, 그 외에는 false***          |
| **!=**  |    ***값이 다름***     | `a != b`  | ***a 값과 b 값이 다르면 true, 그 외에는 false***          |
| **===** | ***값과 타입이 같음*** | `a === b` | ***a와 b의 값과 타입이 같으면 true, 그 외에는 false***    |
| **!==** | ***값과 타입이 다름*** | `a !== b` | ***a와 b의 값과 타입이 다르면 true, 그 외에는 false***    |
|  **<**  |       ***작음***       |  `a < b`  | ***a 값이 b 값보다 작으면 true, 그외에는 false***         |
|  **>**  |        ***큼***        |  `a > b`  | ***a 값이 b 값보다 크면 true, 그 외에는 false***          |
| **<=**  |   ***작거나 같음***    | `a <= b`  | ***a 값이 b 값보다 작거나 같으면 true, 그 외에는 false*** |
| **>=**  |   ***크거나 같음***    | `a >= b`  | ***a 값이 b 값보다 크거나 같으면 true, 그 외에는 false*** |

- **동일(==) 연사자와 일치(===)연산자**는 변수가 같은지를 판정하는 연산자이지만 의미가 다르다.
- **비동일(!=)** 연산자는 동일 연산자와 부정이며 **불일치(!==)**연산자는 일치 연산자의 부정이다.



> **1) 동일 연산자**

`동일(==)`연산자는 좌변과 우변의 피연산자가 같은지를 판별한다.



***1-1) 좌우 피연산자의 타입이 같을 때***

- 값이 같으면 `true`, 다르면 `false`로 판정한다. 
  - 값의 의미는 변수의 데이터 타입에 따라 달라진다.  **원시 타입의 변수의 값과 객체 타입 변수의 값이 다르기 때문에**
  - 원시타입 변수의 값은 **데이터 그 자체**이지만, 객체 타입 변수의 값은 **객체의 참조**라는 점이 다르다.

 ***EX1-1) 배열 두 개가 같은 배열인지 판별***

```javascript
var a = [1,2,3];
var b = [1,2,3];
var c = a;
console.log(a==b); // → false
console.log(a==c); // → true

/* 
a와 b에는 모두 배열의 참조가 저장되어 있다. 각 배열에는 같은 데이터가 담겨 있지만,
메모리에서 차지하는 위치가 다르기에 a값과 b값이 다르다고 판정한다.
*/
```



***1-2) 좌우 피연산자의 타입이 다를 때***

- 두 피 연산자가 가은 타입이 되도록 타입을 변환한 다음에 다음 규칙에 따라 동일한지 판별한다.
  - **`underfined`**와 **`null`**은 ***같은 것***으로 친다.
  - 한 쪽이 **숫자**고 다른 한 쪽이 **문자열**이면 **문자열을 숫자로 변환**해서 비교한다.
  - 둘 중에 한 쪽이 논리값이면 true는 `1`, false는` 0`으로 변환해서 비교한다.
  - 한 쪽이 **`객체`**고 다른 한 쪽이 **`숫자`** 또는 **`문자열`**이면 객체를 **`toString`**이나 **`valueOf `**메서드를 사용해서 **`원시 타입`**으로 변환한 다음에 비교한다.
  - 앞의 규칙에서 벗어나면 모두 **`'같지 않음'`**으로 판정한다.

```javascript
null == underfined // → true
1 == "1" // → true
"0xff" == 255 // → true
true == 1 // → true
true == "1" // → true
(new String("a")) == "a" // → true
(new Number(2)) == 2 // → true
[2] == 2 // → true
```



> **2) 일치 연산자**

- `일치(==)`연산자는 피연산자를 평가한 후에 타입을 변환하지 않은 상태의 두 값을 엄격하게 비교한다. 
- **타입**과 **값**이 모두 같으면 **같다**고 판정하고 그렇지 않으면 **같지 않다**라고 판정한다.
  - 단, `NaN`은 같지 않다고 판정
  - `NaN` === `NaN` // → false
  - `NaN`은 x !== x 를 만족하는 유일한 값임을 확인할 수 있다. 변수 값이 `NaN`인지는 `isNaN` 함수로도 확인할 수 있다.
  - **동일 연산자**의 사용 예에서는 **`true`** 였던 표현식이 **일치 연산자**에서는 모두 **`false`**가 된다.

```javascript
null === underfined // → false
1 === "1" // → false
"0xff" === 255 // → false
true === 1 // → false
true === "1" // → false
(new String("a")) === "a" // → false
(new Number(2)) === 2 // → false
[2] === 2 // → false
```

|  연산자  |     뜻     |     예제     | 예제의 뜻                                           |
| :------: | :--------: | :----------: | --------------------------------------------------- |
|  **&&**  | **논리곱** |  **a && b**  | a와 b가 모두 true면 true, 그 외에는 false           |
| **\|\|** | **논리합** | **a \|\| b** | a와 b 중 하나라도 true면 true, 모두가 false면 false |
|  **!**   |  **부정**  |    **!a**    | a가 true면 false, false면 true                      |

```javascript
x > 0 && y > 0 // x와 y가 모두 참일 때만 true
x > 0 || y > 0 // x와 y 중 하나라도 참이면 true, 모두가 0 이하면 false
!(a && b) // a 또는 b가 거짓 (!가 && 보다 우선순위가 높으므로 소괄호)
```

## _5.5 비트 연산_

비트 연산자는 데이터 통신이나 이진 파일을 처리할 때 자주 사용된다.



### _5.5.1 비트연산_

- **비트연산** : 2진수 숫자의 자리별 값(비트 값)을 다루는 연산
- 비트 연산자는 피연산자를 부호 있는 `32비트 정수`로 변환해서 처리한다.
- 비트 연산자
  - 비트 논리 연산자
  - 비트 시프트 연산자
- 모든 비트 연산자에는 `대입 연산자`를 사용할 수 있다.





### _5.5.2 비트 논리 연산자_

- 비트 논리 연산자는 비트 값이 `0`이면 `false`, 비트 값이 `1`이면 `true`로 평가하는 연산자
  - 논리곱(^), 논리합(v),배타적 논리합(+), 논리 부정(~)

| 연산자 |           뜻            |     예제     |   예제의 뜻    |
| :----: | :---------------------: | :----------: | :------------: |
|   &    |    비트 논리곱(AND)     | 105 & (-91)  |  33 =00100001  |
|   \|   |     비트 논리합(OR)     | 105 \| (-91) |  -19=11101101  |
|   ^    | 비트 배타적 논리합(XOR) |  105^(-91)   |  -53=11001100  |
|   ~    |   비트 논리 부정(NOT)   |     ~05      | -106=10010110※ |

※ 자바스크크립트에서는 음의 정수를 `보수`를 사용해서 표현한다. 즉, a의 보수를 `~a+1`이라고 표현한다.

이 공식에 따라 ~105 = -105 - 1 = -106이 된다.





## _5.6 기타 연산_

|     연산자     | 뜻                                                     |

### _5.6.1 typeof 연산자_

- `typeof `연산자는 **단항 연산자**이며 피연산자의 데이터 타입을 뜻하는 ***문자열을 반환한다.***

```javascript
var s = "ABC";
console.log(typeof s); // string
```

- `typeof`연산자는 함수 이외의 객체에 대해 모두 `"object"`를 반환하므로 객체 유형은 파악할 수 없다.
  - 객체 유형을 조사할 때는 `instanceof`연산자와 객체의 `constructor` 프로퍼티를 사용한다.

***EX) typeof연산자의 반환값***

|      데이터      |            데이터 예시            | typeof 연산자의 반환값 |
| :--------------: | :-------------------------------: | :--------------------: |
|     숫자.NaN     |           12, 3.14, NaN           |        "number"        |
|      문자열      |        "abc", "안녕하세요"        |        "string"        |
|      논리값      |            true, false            |       "boolean"        |
| 정의되지 않은 값 |            underfined             |      "underfined"      |
|     null 값      |               null                |        "object"        |
|       심벌       |          Symbol("heart")          |        "symbol"        |
|  함수 외의 객체  | [1,2,3], String("abc"), Number(1) |        "object"        |
|       함수       |     function(x){return ++x;}      |       "function"       |



### _5.6.2 조건 연산자_

- **`조건(:?)`** 연산자는 주어진 조건의 참과 거짓에 따라 값을 선택한다.

```javascript
var parity = (a % 2 == 0)? "짝수" : "홀수"; 
// a % 2 == 0 는 짝수를 의미한다.
```



***EX) if/else 문을 사용***

```javascript
if(a % 2 == 0){
    parity = "짝수";
}else{
    parity = "홀수";
}
```



### _5.6.3 쉼표 연산자_

- **`쉼표(,)`** 연산자는 이항 연산자로 **`왼쪽 피연산자`**를 평가하고 **`오른쪽 피연산자`**를 평가한 이후에 마지막으로 **`오른쪽 끝 피연산자`**의 값을 반환한다.

***EX)***

```javascript
i = 0, sum = 0, product=1;
i = 0; sum = 0; product=1;

for(var i=1, sum=0; i<=10; i++){
    sum += i;
}
```

- for 문의 소괄호 안에는 문장을 세 개만 넣을 수 있지만,  쉽표 연산자를 사용하여 더 많은 문장을 실행 할 수 있다.



### _5.6.4 eval함수_

- `eval`은 **함수**일 뿐 연산자는 아니다.
- `eval`함수는 문자열 단 하나만을 인수로 받아서 자바스크립트 코드로 해석한다.





## _5.7 명시적 타입 변환_

### _5.7.1 숫자를 문자열로 변환하기_

> **1) 숫자 + 문자열**

- 숫자와 문자열을 `+` 연산자로 연결하면 ***숫자의 타입이 문자열로 바뀐다.***

```javascript
10 + "cookies" // → "10cookies"
100 + "" // → "100" : 숫자에 빈 문자열을 더해 숫자 타입을 문자열로 바꿀 수 있음
("0000" + 12).slice(-4) // → "0012" : 앞 부분에 0을 넣어 네 자릿수 정수를 표현하는 문자열로 만듦
```



> **2) Number 객체의 메서드를 활용하는 방법**

|        메서드        | 설명                                                         |
| :------------------: | ------------------------------------------------------------ |
|    ***toString***    | ***숫자를 문자열로 변환한다.*** 인수로 기수(2~32)를 넘기면 진법을 바꿀 수 있다. |
| ***toLocaleString*** | ***숫자를 지역화된 문자열로 변환한다.*** 인수로 다양한 옵션을 줄 수 있다. |
|    ***toFixed***     | 숫자의 소수점 아래 자릿수를 지정한 문자열로 변환한다.        |
| ***toExponential***  | 숫자의 소수점 아래 자릿수를 지정한 문자열로 변환하되 지수를 함께 표시한다. |
|  ***toPrecision***   | ***숫자를 유효 숫자가 지정된 문자열로 변환한다.*** 단, 유효 숫자가 정수부의 자릿수보다 작을 때는 지수로 표시한다. |

***EX) 사용 예시***

```javascript
var n = 26;
n.toString()		// → "26" : 인수를 지정하지 않으면 10진수 문자열로 변환
n.toString(2)		// → "11010" : 2진수 문자열로 변환
n.toString(16)		// → "1a" : 16진수 문자열로 변환
(26).toString(16)	// → "1a" : 숫자에서 바로 메서드를 사용하고자 할 때는 소괄호로 묶는다

var x = 1234.567;
x.toString()		// → "1234.567"
x.toString(16)		// → "4d2.9126e978d5"
x.toFixed(0)		// → "1235"
x.toFixed(2)		// → "1234.57"
x.toFixed(4)		// → "1234.5670"
x.toExponential(3)	// → "1.235e+3"
x.toPrecision(3)	// → "1.23e+3"
x.toPrecision(6)	// → "1234.57"
```



> **3) String 함수를 활용하는 방법**

- `String` 생성자 앞에 `new` 연산자를 붙이면 `String` 객체를 생성하는 함수로 사용할 수 있지만, `new`연산자를 붙이지 않으면 일반적인 함수로 활용할 수 있다.

- String 함수에는 모든 데이터 타입을 **`문자열`**타입으로 바구는 기능이 있다.

```javascript
String("ABC")			// → "ABC"
String(true)			// → "true"
String(false)			// → "false"
String(NaN)				// → "NaN"
String(null)			// → "null"
String(underfined)		// → "underfined"
String({x:1, y:2})		// → "[object Object]"
String({x:1, y:2, z:3})	// → "[object Object]"
String([1,2,3])			// → "1,2,3"
```



### _5.7.3 논리값으로 변환하기_

- 모든 값을 논리 값으로 바꾸는 방법
  - !!x			
  - Boolean(x)

`!`연산자는 논리 타입이 아닌 값의 타입을 논리 타입으로 바꾼다.

그다음에 `!`를 하나 더 붙여서 참과 거짓을 뒤바꾸고 있다.

이로써, `!!`만을 사용하여 값의 타입을 논리 타입으로 바꿀 수 있다.
